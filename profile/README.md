## Description
**고양이 커뮤니티 '야옹타운'입니다 !** <br>
- 야옹타운은 자유롭게 고양이 사진을 공유할 수 있는 커뮤니티입니다.
- 회원가입, 로그인만 하면 누구나 자유롭게 이용 가능합니다.
- 마음에 드는 게시글엔 댓글을 작성하거나 좋아요를 누를 수 있습니다.
